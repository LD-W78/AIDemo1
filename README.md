# Sketch2Art - 我画你生成

一个创意绘画Web游戏，用户随意涂鸦，AI自动补全成艺术作品。

## 功能特点

- 🎨 **自由绘画**：支持鼠标和触摸绘制
- ✨ **AI补全**：6种艺术风格可选（水彩、油画、像素、赛博、素描、抽象）
- 🖌️ **画笔调节**：可调节画笔大小
- ↩️ **撤销功能**：支持撤销上一步操作
- 💾 **下载作品**：保存生成的艺术作品
- 📱 **响应式设计**：支持桌面和移动设备

## 艺术风格

1. **水彩** - 柔和透明的水彩效果
2. **油画** - 厚重质感的油画风格
3. **像素** - 复古像素艺术
4. **赛博** - 霓虹赛博朋克风格
5. **素描** - 精细铅笔素描
6. **抽象** - 几何抽象艺术

## 技术栈

- React 18 + TypeScript
- Tailwind CSS
- HTML5 Canvas API
- Vite

## 开始使用

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build
```

## 游戏玩法

1. 在左侧画布上随意涂鸦
2. 选择喜欢的艺术风格
3. 点击"生成艺术作品"
4. 欣赏AI补全的结果
5. 下载保存你的作品

## 设计亮点

- 深色主题配合霓虹渐变背景
- 毛玻璃效果和流畅动画
- 直观的单页应用设计
- 沉浸式的绘画体验
